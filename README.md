# My Stego Project

## 📌 Description
This project is based on steganography, which is the technique of hiding secret data inside images or other media files.

## 🚀 Features
- Hide secret messages inside images
- Extract hidden messages
- Simple and user-friendly implementation

## 🛠️ Technologies Used
- Python
- numpy
- cryptography
- haslib


## ▶️ How to Run
1. Clone the repository
2. Run the Python file
3. Follow instructions in the program

## 📂 Project Structure
- main.py → Main code
- images/ → Input images
- output/ → Stego images

## 🎯 Purpose
This project is developed for learning and demonstrating data hiding techniques.
